1. 要引入proto 文件

    import './proto'

2. service 定义不能引入null,要使用null，可以使用一个空对象

        service attributeItem{
            rpc getAll(null) returns (rtn);  // 这里使用null或者undefined 都会报错
        }

        解决方案：

        message Empty{}

        service attributeItem{
            rpc getAll(Empty) returns (rtn);
        }
